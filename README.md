# briannadoubt's Claude Code Plugins

A collection of Claude Code plugins for development workflows.

## Install

```bash
/plugin marketplace add briannadoubt/claude-marketplace
```

## Available Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| [apple-docs](https://github.com/briannadoubt/AppleDocsTool) | Apple development tools - simulator UI, project analysis, build/test | `/plugin install apple-docs@briannadoubt` |

## Adding This Marketplace

Once added, browse available plugins:
```bash
/plugin
# Navigate to "Marketplaces" tab to see plugins
```

Or install directly:
```bash
/plugin install <plugin-name>@briannadoubt
```
